# Sudoku

Browser sudoku, no accounts and no ads. Play it here: https://yoonzky.github.io/sudoku/

Single html file, no dependencies. Progress and settings are stored in localStorage.

## Features

- digit popup right at the cell (double or right click)
- draft mode for testing a hypothesis, commit or roll back in one tap
- remaining digit counters on the numpad
- full keyboard control (Space for notes, H for hint, arrows to move)
- 9 themes, 6 languages, works offline

## Run locally

Just open `index.html` in a browser.
